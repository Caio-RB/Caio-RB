### Projeto principal – Kruks
**Kruks** é o meu hub multimídia de indexação de conteúdo indie. O objetivo é centralizar, catalogar, divulgar e servir obras independentes com uma experiência semelhante à de um serviço de streaming, de forma totalmente gratuita.
- Player de vídeo integrado
- Integração de comunidades
- Guias para artistas independentes
- Centralização do acesso à cultura indie
